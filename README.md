# Task Manager App

A task management web application built using React, Firebase Authentication, Cloud Firestore, and Tailwind CSS. Users can register, log in, and manage their personal tasks with categories, priorities, deadlines, and completion tracking.

## Features

-- User Registration & Login using Firebase Authentication
-- User-specific task storage with Firestore
-- Add new tasks
-- Edit existing tasks
-- Delete tasks
-- Mark tasks as completed / undo
-- Filter tasks by category
-- Task counter
-- Responsive user interface
-- Deployed on Vercel

## Tech Stack
• React.js
• Vite
• Tailwind CSS
• Firebase Authentication
• Cloud Firestore
• Vercel
